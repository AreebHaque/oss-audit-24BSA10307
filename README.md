## Scripts Overview

### Script 1: System Identity Report
Displays system information such as:
- Linux distribution  
- Kernel version  
- Current user  
- System uptime  
- Date and time  
- License information  

---

### Script 2: FOSS Package Inspector
- Checks if a package is installed  
- Displays version and status  
- Uses case statement to describe software  

---

### Script 3: Disk and Permission Auditor
- Scans important directories  
- Displays permissions, owner, and size  
- Checks configuration directory  

---

### Script 4: Log File Analyzer
- Reads log file line by line  
- Counts keyword occurrences  
- Displays last 5 matching lines  
- Handles missing file errors  

---

### Script 5: Open Source Manifesto Generator
- Takes user input  
- Generates a personalized manifesto  
- Saves output to a text file
